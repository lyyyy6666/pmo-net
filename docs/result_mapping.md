# Result Mapping

Final experimental result files and manuscript table CSVs are not included in this public release repository.

This document is intentionally limited to release-scope guidance:

- The repository provides model source code, dataset-format instructions, graph construction utilities, configuration files, and training/evaluation scripts.
- Checkpoints, raw prediction arrays, logs, final `best_metrics.csv`/`test_metrics.csv` files, and table-level final result CSVs are excluded.
- Reported numerical results should be taken from the manuscript and its approved research data archive, not from this GitHub source-code release.

If result files are shared separately, keep a separate mapping outside this source-code repository that records the original experiment directory, metric file, split, and manuscript table row.
